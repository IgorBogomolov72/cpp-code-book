## multithreaded_telemetry_server
### (Проект в разработке)

Многопоточный сервер телеметрии. 
* Сервер на C++, который одновременно обслуживает множество подключений от устройств.
* Принимает данные от устройств по протоколу TCP.
* Ведёт учёт подключённых устройств и их последних показаний.
* Позволяет отправлять команды обратно на устройства.
* Масштабируется за счёт многопоточности.
* Сохраняет историю данных в базу SQLite.

Для демонстрации работы сервера создаются эмуляторы устройств — отдельные программы (или потоки в тестовом окружении), которые подключаются к серверу и шлют случайные данные, имитируя показания датчиков. Эмуляторы запускаются в многопоточном режиме, чтобы проверить производительность сервера под нагрузкой.
